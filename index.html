Algorithm Information

Dijkstra's Algorithm

Dijkstra's algorithm is a weighted algorithm that guarantees the shortest path. It works by visiting nodes in order of increasing distance from the start node.

Time Complexity: O((V + E) log V) where V is the number of vertices and E is the number of edges.

Space Complexity: O(V)

How to Use

1 Select an Algorithm

Choose from Dijkstra's, A*, BFS, or DFS algorithms. Each has different strengths and use cases.

Weighted algorithms (Dijkstra's and A*) consider node weights, while unweighted algorithms (BFS and DFS) do not.

2 Customize Your Grid

Click and drag to draw walls that block the path.

When weighted nodes are enabled, press 'W' while clicking to create weighted nodes.

Drag the start (green) and end (red) nodes to reposition them.

3 Run the Visualization

Click the "Start Visualization" button to see the algorithm in action.

Adjust the animation speed slider to control how fast the visualization runs.

Watch as the algorithm explores the grid and finds the shortest path!
